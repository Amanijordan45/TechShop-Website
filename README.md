# 🛒 TechShop – Multi-Page E-commerce Website

![TechShop Banner](images/hero.png)

**TechShop** is a modern, responsive multi-page e-commerce website template for selling IT products and technology gadgets.
It is built using **HTML, CSS, and JavaScript**, and demonstrates a clean design with interactive UI elements like FAQ accordion, testimonials, and a contact form.

This project is perfect as a **portfolio project for beginner web developers** learning front-end development.

# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/TechShop.git
```

Navigate into the project folder:

```bash
cd TechShop
```

Open the project:

```
Open index.html in your browser
```

---

# 🌍 Deploy on GitHub Pages

1. Go to your repository on **GitHub**
2. Click **Settings**
3. Select **Pages**
4. Choose:

```
Branch: main
Folder: /root
```

5. Save and GitHub will generate your website link.

Example:

```
https://your-username.github.io/TechShop/
```
